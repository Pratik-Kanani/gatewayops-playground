# GatewayOps Playground

Interactive showcase environment for GatewayOps MCP.

GatewayOps Playground demonstrates governed AI execution flows for payment operations using simulated gateway integrations and scenario-driven workflows.

## Goals

- Interactive execution walkthroughs
- Tool orchestration visualization
- Policy engine demonstrations
- Simulated payment workflows
- Architecture showcase

## Planned Scenarios

- Payment diagnostics
- Payment link generation
- Policy approval flows
- Runtime protection demos

## Stack

Frontend:
- Next.js
- TypeScript
- Tailwind

Backend:
- GatewayOps MCP
